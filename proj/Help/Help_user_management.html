User Management

Overview

This is module is accessible only by the system administrator. This module allows the administrator to edit (ie. update and insert) the users, persons, and family_doctor tables. This module is accessible from the home screen by clicking on the link titled User Management:

How to Edit Users Table

Insert

The users table can be edited by selecting the link titled 'Users' from the user management page.

Other

The rest of the info can be edited through the 'Update Personal Info' form. Simply enter the new info in the desired field and press the 'Update Personal Info' button. Any fields that are left empty will not be updated. All of your information, except the password, will be displayed at the top of the page. If there are any errors they will be printed beneath the form.
