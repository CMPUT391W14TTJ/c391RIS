User Management

Overview

This is module is accessible only by the system administrator. This module allows the administrator to edit (ie. update and insert) the users, persons, and family_doctor tables. This module is accessible from the home screen by clicking on the link titled User Management:

How to Edit Users Table

Insert

The users table can be added to by selecting the link titled 'Users' from the user management page. From here, information can be added to the form labelled "Insert a New User" to insert the user into the table. Missing information will cause a flag to show you what information is required for a successful insert.

Update

The users table can be edited by selecting the link titled 'Users' from the user management page. From here, information can be revised with the form labelled "Update an Existing User" to change a row in the table. By entering the user's old ID, any other columns can optionally be added to alter just that column in the table.

How to Edit Persons Table

Insert

The persons table can be altered to by selecting the link titled 'Persons' from the user management page. From here, information can be added by not supplying a person id. This will cause a new person to be automatically inserted with the information provided.

Update

The users table can be altered by selecting the link titled 'Persons' from the user management page. From here, information can be revised with by supplying a person id. This will cause an existing user to be updated with the information provided.

How to Edit Family Doctor Table

Insert

The family_doctor table can be added to by selecting the link titled 'Family Doctor' from the user management page. From here, information can be added to the form labelled "Insert" to insert a new doctor/patient into the table. Missing information will cause a flag to show you what information is required for a successful insert.

Update

The family_doctor table can be updated by selecting the link titled 'Family Doctor' from the user management page. From here, information can be revised by using the form labelled "Update" to edit an existing new doctor/patient entry in the table. Missing information will cause the blank entry to remain constant.
